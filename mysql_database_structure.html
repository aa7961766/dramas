MySQL数据库结构详情

完整的数据库表结构定义，包含所有表、字段、数据类型及约束条件

数据库说明

本数据库用于存储和管理短剧网站的所有数据，包括用户信息、剧集内容、分类、评论互动等。结构设计遵循规范化原则，确保数据完整性和一致性。

数据库包含四大类表：基础分类表、核心内容表、交互表和扩展功能表，各表之间通过外键关联，形成完整的数据关系网络。

基础分类表

drama_categories (剧集分类表)

基础表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 分类唯一标识
name VARCHAR(100) NOT NULL UNIQUE 分类名称（如 "喜剧""悬疑"）
created_at DATETIME DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 最后更新时间（自动维护）

article_categories (文章分类表)

基础表
字段名 数据类型 约束 / 默认值 说明
与 drama_categories 表结构完全相同
id INTEGER PRIMARY KEY AUTO_INCREMENT 分类唯一标识
name VARCHAR(100) NOT NULL UNIQUE 分类名称（如 "攻略""资讯"）
created_at DATETIME DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 最后更新时间（自动维护）

核心内容表

users (用户表)

核心表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 用户唯一标识
username VARCHAR(50) NOT NULL UNIQUE 用户名（登录用）
password VARCHAR(255) NOT NULL 密码哈希值（禁止明文）
email VARCHAR(100) UNIQUE 邮箱（用于找回密码）
role VARCHAR(20) DEFAULT 'user' 角色（'user' 普通用户 /'admin' 管理员）
status VARCHAR(20) DEFAULT 'active' 状态（'active' 正常 /'banned' 封禁）
created_at DATETIME DEFAULT CURRENT_TIMESTAMP 注册时间
updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 信息更新时间

dramas (剧集表)

核心表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 剧集唯一标识
name VARCHAR(255) NOT NULL 剧集名称
link TEXT 播放链接
description TEXT 剧情描述
category_id INTEGER FOREIGN KEY (drama_categories.id) 关联的分类 ID
image_path TEXT 封面图片路径
published_at DATETIME DEFAULT CURRENT_TIMESTAMP 发布时间
updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间
is_deleted TINYINT DEFAULT 0 软删除标记（0 = 正常，1 = 已删除）

articles (文章表)

核心表
字段名 数据类型 约束 / 默认值 说明
与 dramas 表结构基本相同，不含 link 字段，增加 content 字段
id INTEGER PRIMARY KEY AUTO_INCREMENT 文章唯一标识
name VARCHAR(255) NOT NULL 文章标题
description TEXT 文章摘要
category_id INTEGER FOREIGN KEY (article_categories.id) 关联的分类 ID
image_path TEXT 封面图片路径
content TEXT NOT NULL 文章正文内容
published_at DATETIME DEFAULT CURRENT_TIMESTAMP 发布时间
updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间
is_deleted TINYINT DEFAULT 0 软删除标记（0 = 正常，1 = 已删除）

交互表

comments (评论表)

交互表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 评论唯一标识
content TEXT NOT NULL 评论内容
created_at DATETIME DEFAULT CURRENT_TIMESTAMP 评论时间
article_id INTEGER FOREIGN KEY (articles.id) 关联的文章 ID（可为 NULL）
drama_id INTEGER FOREIGN KEY (dramas.id) 关联的短剧 ID（可为 NULL）
user_id INTEGER NOT NULL, FOREIGN KEY (users.id) 评论用户 ID
is_deleted TINYINT DEFAULT 0 软删除标记

likes (点赞表)

交互表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 点赞记录唯一标识
content_type VARCHAR(20) NOT NULL 内容类型（'article'/'drama'）
content_id INTEGER NOT NULL 关联的内容 ID
user_id INTEGER NOT NULL, FOREIGN KEY (users.id) 点赞用户 ID
created_at DATETIME DEFAULT CURRENT_TIMESTAMP 点赞时间
unique_like 复合索引 (content_type, content_id, user_id) 避免同一用户重复点赞

扩展功能表

tags (标签表)

扩展表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 标签唯一标识
name VARCHAR(50) NOT NULL UNIQUE 标签名称（如 "热门""新上线"）
created_at DATETIME DEFAULT CURRENT_TIMESTAMP 创建时间

article_tags (文章-标签关联表)

扩展表
字段名 数据类型 约束 / 默认值 说明
article_id INTEGER NOT NULL, FOREIGN KEY (articles.id) 关联的文章 ID
tag_id INTEGER NOT NULL, FOREIGN KEY (tags.id) 关联的标签 ID
主键 复合主键 (article_id, tag_id) 避免重复关联

drama_tags (短剧-标签关联表)

扩展表
字段名 数据类型 约束 / 默认值 说明
与 article_tags 表结构完全相同
drama_id INTEGER NOT NULL, FOREIGN KEY (dramas.id) 关联的短剧 ID
tag_id INTEGER NOT NULL, FOREIGN KEY (tags.id) 关联的标签 ID
主键 复合主键 (drama_id, tag_id) 避免重复关联

统计与配置表

statistics (统计信息表)

系统表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 统计记录唯一标识
content_type VARCHAR(20) NOT NULL 内容类型（'article'/'drama'）
content_id INTEGER NOT NULL 关联的内容 ID
click_count INTEGER DEFAULT 0 点击量（需调用 record_click () 更新）
comment_count INTEGER DEFAULT 0 评论数（触发器自动更新）
like_count INTEGER DEFAULT 0 点赞数（触发器自动更新）
update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 最后统计更新时间

search_keywords (搜索关键词表)

系统表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 关键词唯一标识
keyword VARCHAR(100) NOT NULL UNIQUE 搜索关键词
count INTEGER DEFAULT 1 搜索次数
last_searched DATETIME DEFAULT CURRENT_TIMESTAMP 最后搜索时间

settings (网站设置表)

系统表
字段名 数据类型 约束 / 默认值 说明
id INTEGER PRIMARY KEY AUTO_INCREMENT 设置记录唯一标识
site_name VARCHAR(255) NOT NULL 网站名称
site_url VARCHAR(255) NOT NULL 网站域名
site_keywords TEXT 网站 SEO 关键词
site_description TEXT 网站描述
updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 设置更新时间
